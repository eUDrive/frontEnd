# 🏎️ eUDrive - Аренда гоночных автомобилей

Премиальный сервис аренды гоночных автомобилей на легендарной трассе **Nürburgring Nordschleife**.

## 🚀 Запуск проекта

### Установка зависимостей
```bash
npm install
```

### Запуск dev сервера
```bash
npm run dev
```

Приложение откроется на `http://localhost:5173` (или другом доступном порту).

### Сборка для продакшена
```bash
npm run build
```

## 🎨 Дизайн

- **Цветовая схема**: Светлая тема с красными акцентами (#FF0000)
- **Шрифт**: Montserrat (Google Fonts) - жирный, округленный
- **Стиль**: Спортивный, динамичный, современный

## 🛠️ Технологии

- **React 18** + TypeScript
- **React Router** - маршрутизация
- **Vite** - сборщик
- **CSS Modules** - стилизация

## 📄 Страницы

### 🏠 Главная (/)
- Hero секция с призывом к действию
- Кнопка "Выбрать авто" → переход в каталог

### 🚗 Каталог (/catalog)
- Сетка карточек автомобилей
- Поиск по бренду/модели
- Фильтры по категориям (Sport, GT, Touring)
- 6 автомобилей в наличии

### 📦 Пакеты (/packages)
- Пакет 6 кругов (125 км)
- Пакет 12 кругов (250 км)
- Описание и цены

### ℹ️ О нас (/about)
- Информация о сервисе
- Преимущества
- Контакты
